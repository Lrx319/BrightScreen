# BrightScreen · 明格大屏

> 一个面向初学者的开源数据可视化大屏项目，帮助你从 0 到 1 动手做出自己的数据大屏。

BrightScreen（明格大屏）是一个公开、循序渐进的实战项目，专为想了解「数据可视化大屏」如何诞生的同学而设计。
我们会从最基础的环境搭建开始，一步步带你掌握布局、组件、数据接入与动态刷新的完整流程。

## 项目目标

- 理解数据大屏的整体结构与设计思路
- 掌握从静态页面到动态数据可视化的实现过程
- 通过可运行的示例，边学边做，最终能独立搭建自己的大屏
